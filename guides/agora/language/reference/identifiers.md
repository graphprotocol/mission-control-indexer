# Identifiers
An Identifier starts with an ASCII letter or an underscore, and is followed by zero or more ASCII letters, numbers, and/or underscores.

```
# Valid identifiers:
BE
_KIND_
_2
One
another

# Not identifiers:
42
Fun!
Î»
```
